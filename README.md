# Furcord [Alpha]
Discord mod but made by a furry.

## Installation

Install dependencies: `npm i`

Build Source: `npm run build`

Inject: `npm run inject [-- --ptb|canary]`

## Uninstallation

Uninject: `npm run uninject [-- --ptb|canary]`
